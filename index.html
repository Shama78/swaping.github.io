Swapping of 2 numbers in C

Swapping two number in C programming language means exchanging the values of two variables. Suppose you have two variable a & b. Value of a is 5 & value of b is 4. So, after swapping the value of a will become 4 & value of b will become 5. In this blog will understand how to swap two variables in C. Let's go for the examples



Swapping example 1

Swapping of 2 numbers

                  //Program to swap 2 number 
                  #include<stdio.h>
                  int main()
                  {
                      int a,b;
                      printf("Enter the 2 numbers\n");
                      scanf("%d %d",&a, &b);
                      a=a+b;
                      b=a-b;
                      a=a-b;
                      printf("value of first number: %d\n",a);
                      printf("value of second number: %d\n",b);
                      return 0;
                  }

Swapping example 2

Swapping of 2 numbers using third variable

                  //Program to swap 2 number using third variable
                  #include<stdio.h>
                  int main()
                  {
                      int a, b, c;
                      printf("Enter the 2 numbers\n");
                      scanf("%d %d",&a, &b);
                      c=a;
                      a=b;
                      b=c;
                      printf("value of first number: %d\n",a);
                      printf("value of second number: %d\n",b);
                      return 0;
                  }
